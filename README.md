# 💡 Uniswap Range Helper

Uniswap V3에서 다른 LP들이 가장 많이 사용한 가격 범위를 실시간으로 보여주는 Chrome 확장 프로그램입니다.

## ✨ 주요 기능

- 📊 **인기 범위 분석**: 최근 5,000블록 기준으로 가장 많이 사용된 가격 범위 표시
- ⚡ **원클릭 적용**: 추천 범위를 클릭 한 번으로 입력 필드에 자동 입력
- 💰 **예상 APY**: 범위 폭과 fee tier를 고려한 대략적인 APY 추정치 제공
- 🎯 **범위 폭 분석**: 집중/중간/넓음 등 범위 효율성 평가
- 🔄 **스마트 가격 변환**: 토큰 순서를 자동으로 감지하여 올바른 가격 표시

## 🖼️ 스크린샷

확장 프로그램을 설치하면 Uniswap 포지션 생성 페이지에서 이런 화면을 볼 수 있습니다:

## 🚀 설치 방법

### Chrome에서 설치

1. 이 저장소를 다운로드하거나 클론합니다:
   ```bash
   git clone https://github.com/kalhintz/uniswap_rader.git
   ```

2. Chrome 브라우저를 열고 `chrome://extensions/` 로 이동합니다

3. 우측 상단의 **개발자 모드**를 활성화합니다

4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다

5. 다운로드한 폴더를 선택합니다

6. 설치 완료! 🎉

## 📖 사용 방법

1. [Uniswap](https://app.uniswap.org)에 접속합니다

2. **Pool** → **New Position** 을 클릭하여 포지션 생성 페이지로 이동합니다

3. 토큰 페어와 fee tier를 선택합니다

4. 확장 프로그램이 자동으로 인기 범위를 분석하여 표시합니다

5. 원하는 범위의 **[적용]** 버튼을 클릭하면 자동으로 입력됩니다

## 🔧 작동 원리

1. **데이터 수집**: Ethereum RPC를 통해 최근 5,000블록의 Mint 이벤트 수집
2. **범위 분석**: Tick 범위를 가격으로 변환하고 사용 빈도 계산
3. **가격 변환**: 토큰 순서를 감지하여 사용자가 선택한 페어에 맞게 가격 표시
4. **APY 추정**: Fee tier와 범위 폭을 고려한 대략적인 APY 계산

## ⚠️ 주의사항

- **APY는 대략적인 추정치**입니다. 실제 수익은 거래량, 가격 변동 등에 따라 크게 달라질 수 있습니다
- Uniswap V3의 **Impermanent Loss** 위험을 항상 고려하세요
- 이 도구는 참고용이며, 투자 결정은 본인의 책임입니다
